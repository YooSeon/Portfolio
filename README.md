### 1. TitleScene

   #### 주요기능
   
   - SkyBox 회전을 이용한 BackGround 구현
   - Image Rotate을 이용한 회전 이미지 구현
   - 효과음이 끝났을 경우 Scene 전환 구현
   - GameMgr을 이용한 TitleScene 및 SelectScene 제어
